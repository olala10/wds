Detailed Description

Klasa reprezentująca nowe okno wykresów aplikacji.

Klasa Diagram dziedzicząca po klasie QDialog, reprezentuje nowe okno aplikacji. Zawiera deklarację interfejsu użytkownika (UI) oraz prywatne pola i metody.
