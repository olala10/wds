Detailed Description

Klasa reprezentująca nowe okno aplikacji.

Klasa NewWindow dziedzicząca po klasie QWidget, reprezentuje nowe okno aplikacji. Zawiera deklarację interfejsu użytkownika (UI) oraz prywatne pola i metody.
